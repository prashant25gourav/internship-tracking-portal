# Internship Tracking Portal

## 📌 Overview

The Internship Tracking Portal is a full-stack web application designed to simplify internship management for students, faculty, and companies.

The system allows students to:
- register and login
- browse internships
- apply for internships
- track application status
- upload internship reports

Faculty/Admins can:
- monitor applications
- manage internship workflows
- review uploaded reports

The project demonstrates:
- REST API development
- Database integration
- File upload & management
- Frontend-backend integration
- Team collaboration using Git & GitHub
